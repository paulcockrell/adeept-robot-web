# RoboConsole

A web server for the adeept aws robot

## Development

### Backend

```bash
watchexec -r -e clj -- clojure -M -m myapp.backend.main
```

### Frontend

```bash
clojure -M:dev
```

### Docker

Start everything with Docker + docker-compose

```bash
docker-compose up
```
